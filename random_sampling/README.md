# Random Sampling

This project is a [typescript](https://www.typescriptlang.org/) [deno](https://deno.com/) codebase for running demo of Random Sampling of Unique (x,y) points without replacement

## How to

### Sample

```
% yarn run sample
```

will random sample (x,y) coordinates using a variation of Floyd's Random Sampling without replacement and the Division Theorem to 
random sample unique (x,y) coordindates without replacement
